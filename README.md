## How to install
You need **Eclipse Mars or Luna** to install this theme.

1. Drag [<img src="http://marketplace.eclipse.org/misc/installbutton.png">](http://marketplace.eclipse.org/marketplace-client-intro?mpc_install=339851) on to the Toolbar of your Eclipse 4 to install **Jeeeyul's Eclipse Themes**.
2. After installation, Choose `Jeeeyul's themes - Custom Theme` from `Preferences > General > Appearance`.

If you don't have the **Market Place Client** in your Eclipse, please refer to the [Alternative Installation Guide](https://github.com/jeeeyul/eclipse-themes/wiki/Alternative-Install).

### Eclipse 4.2 to 4.3
Since there were critical API changes on SWT and CSS, Eclipse Luna and below versions are not supported now. 
However, you can install **Jeeeyul's Eclipse Themes v2.2.x (Old Version)** for Eclipse 4.2 to 4.3. Read the details in the [Alternative Installation Guide](https://github.com/jeeeyul/eclipse-themes/wiki/Alternative-Install).

## Customize
